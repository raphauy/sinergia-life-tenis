# Research

Investigaciones previas a una feature: comparativas de tecnologías/proveedores, análisis de costos, estado del arte, hallazgos que fundamentan decisiones del PRP.

Se referencia desde el PRP de la feature (`docs/PRPs/<feature>-prp.md`) y desde el glosario cuando introduce términos nuevos.

Convención: `<tema>.md` (ej: `elo-ranking.md`, `cobros-saas.md`). Para investigaciones extensas, podés usar la skill `/deep-research`.
